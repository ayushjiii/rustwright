# Security Policy

Rustwright is alpha software. The supported version for security review is the
current alpha on the default branch.

Please do not report suspected vulnerabilities in public GitHub issues. Instead,
contact the maintainers privately through the Skyvern Discord:
https://discord.gg/fG2XXEuQX3.

Include a clear description, reproduction steps, affected versions or commits,
and any relevant logs or proof-of-concept details. We will acknowledge reports
as maintainers are available and coordinate fixes in good faith.

No bug bounty or reward program is promised.
